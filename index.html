Basic HTML tags:

Headings (<h1>-<h6>)

  • Headings are used to define titles and subtitles on a webpage.
  • <h1> is the largest and <h6> is the smallest.
<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Smaller Subheading</h3>

Paragraph (<p>)

The <p> tag is used to define a paragraph of text. It automatically adds line breaks before and after the content.
This is a paragraph of text.

Hyperlinks(<a href="URL">)

<a href="https://www.example.com">Visit Example</a>
https://www.youtube.com/

Images(<img src="images.jpg" alt="Description">)

The <img> tag is used to display images on a webpage.
  • The src attribute specifies the image file path.
  • If the image cannot be loaded due to an error(broken link, network issue), the alt text will be displayed instead.
<img src="../images/pic.jpg" alt="Go up one level">

In HTML, file paths specify the location of a file(such as an image, stylesheet, or script). There are two main types of file paths:absolute paths and relative paths.

An absolute path specifies the full URL or complete directory location of a file. It includes the protocol(http/https) and the domanin name.
<img src="https//www.example.com/imgaes/photo.jpg" alt="Example Image">

Use absolute paths when linking to external resources(like online images or stylesheets)

A relative path specifies the location of a file relative to the current HTML document. It does not include the full URL.
<img src="photo.jpg" alt"Image in the same folder">

<img src="images/photo.jpg" alt="Image inside a subfolder">

./ refers to the current directory where the HTML file is located.

../moves one level up to the parent directory of the current file.

Use relative paths for all internal files to ensure portability and better file management.

Lists(<ul>, <ol> and <li>)


Lists help organize content into bullet points or numbered steps.
  • Unordered List(<ul>): Displays bulleted items.
  • Ordered List(<ol>): Displays numbered items.
The <li> tag defines an individual list item within an unordered (<ul>) or ordered (<ol>) list.
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>
In HTML, you can nest formatting tags like <b>(bold), <i>(italic), and <u>(underline) to apply multiple styles to text.
<p>This is<b><i>Bold and Italic</i></b>
In HTML5, the color property is used within the style attribute to define text.

Like this, a red text.

Using a Color Name:
<p style="color:red;">Like this, a red text.</p>
Using a Hexadecimal Color Code:
<p style="color:#ff0000;">Like this, a red text.</p>

Tables(<table>, <tr>, <td>)

An HTML table is used to display data in a strucutred format using rows and columns. It is created using <table> tag, with rows (<tr>) and cells (<tr>) and cells (<td> for data, <th>for headers).
<table border="1">

    <th>Fruit</th>
    <th>Color</th>
  </tr>
  <tr>
    <td>Apple</td>
    <td>Red</td>
  </tr>
  <tr>
    <td>Banana</td>
    <td>Yellow</td>
  </tr>
</table>
